This password strength meter won't work correctly unless the folder is available. If you run the password strength meter simply as a html document, it won't have the ability to compare passwords to the leaked password lists. The other parts functions the same. This was created with Visual studio code, using the live server extension. You can also host a server with the python. Find the directory with the files, type the command "python3 -m http.server 8000", open a webbrowser and access it through http://localhost:8000
